# Supervised Learning - Classification _ WEEK 5 

## Deskription

Project ini merupakan implementasi Supervised Learning (klasifikasi) menggunakan dataset Titanic. Tujuan utamanya adalah 

memprediksi apakah seorang penumpang bertahan hidupatau tidak berdasarkan beberapa fitur seperti usia, jenis kelamin, kelas penumpang, 

dan variabel lainnya. Project ini disusun sebagai bagian dari proses pembelajaran Machine Learning
